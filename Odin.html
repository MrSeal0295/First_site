Одиночные теги

  • base - базовый адрес страницы

  • area - часть карты изображений

  • br - перевод строки

  • col - форматирует одну/несколько колонок таблицы

  • param - параметры плагинов applet и object

  • link - загружает внешние файлы

  • source - внедряет аудио/видео файлы, с помощью тегов audio и video

  • Track - внедряет текстовую информацию

  • wbr - указывает, где нужно перенести длинное слово, если оно не вмещается в окно браузера

  • basefont - форматирует текст для всей страницы

  • embed - внедряет плагины

  • input - элемент формы, может быть кнопкой, текстовым полем, перключателем и др.

  • isindex - поисковая строка

  • keygen - шифровка расшифровка данных

  • meta - мета-теги
